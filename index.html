À propos de moi

Étudiante en BTS Services Informatiques aux Organisations (SLAM) au Lycée Camille Sée de Colmar, je développe des solutions web depuis deux ans, de la maquette jusqu'au déploiement.

GestStages

Déploiement d'une application web en environnement de préproduction

Réalisé en binôme (SLAM + SISR), ce TP consistait à déployer une application PHP/SQL de gestion de stages sur une infrastructure virtualisée Debian, avec deux serveurs spécialisés (SRV-WEB et SRV-BDD) configurés manuellement.

Mission SLAM : déploiement de l'application via SSH/SCP, importation du script SQL, création de l'utilisateur base de données, ajout du fichier de diagnostic PHP, gestion des droits (groupe dev, permissions 775).

Outils : VirtualBox, Debian, Apache2, PHP, MariaDB, SSH, SCP.

AP Transcité

Application web de gestion d'un réseau de transport urbain

Création d'une application web multi-pages connectée à une base de données de réseau de bus. Le projet couvre la modélisation (Looping), l'initialisation des données, l'exploitation par requêtes SQL et le développement de 6 pages PHP dynamiques.

Fonctionnalités : liste des lignes, stations par ligne, guichets automatiques manquants, statistiques de satisfaction par critère, formulaire d'ajout de commentaire.

Outils : WAMP, PHP, SQL, HTML/CSS, Looping.
